## C. 알고리즘 분석실(lab) — 7개의 눈 [⚠️ 핵심 보강 필요]

| 알고리즘 | 재창조 | 현재 |
|---|---|---|
| ① K-means 색 군집화 | 대표색 점 치환 | ✅ (색군집스튜디오) |
| ② 색 히스토그램 | 포스터화 | ❌ |
| ③ 명도·노탄(Notan) | 흑백 명암지도 | ❌ |
| ④ 에지(Sobel) | 선 드로잉 | ❌ |
| ⑤ 중앙값 분할(Median Cut) | 색 요약 비교 | ❌ |
| ⑥ 격자 평균·모자이크 | 픽셀아트 | ⚠️ (렌즈 오버레이만) |
| ⑦ 구도 분석 | 무게중심·삼분할 | ⚠️ |

**색 분석 차트 6종**: 도넛·정렬막대·색공간 산점도·RGB 히스토그램·색상환·명도 톤커브 → ✅ 색군집스튜디오에 구현(7종). lab 단독에는 ❌.
**각 분석마다 해석 메모 → 리포트(부록 A) 자동 취합** → ❌.
**실제 퍼블릭 도메인 명화**(고흐·쇠라·모네·호쿠사이·모나리자·베르메르·클림트·뭉크·칸딘스키·몬드리안) → ❌ (현재 절차적 그라데이션 샘플).

## D. 스튜디오 세부

**색 군집** ✅ — 입력·K-means·차트7·입자·3D·마우스4·마이크·프리셋·내보내기5·윤리리포트.
**데이터 점** ⚠️ — 매핑·레이아웃·A/B ✅ / **데이터 선택 기록(제작문서: 감각먼저→센 것·뺀 것→왜 이 척도→데이터가 놓친 내 삶의 진실)** ❌, Dear Data·데이터 휴머니즘 ⓘ 연결 ❌.
**소리** ✅ — 파형·스펙트럼·특징·매핑·CSV 전송.
**내 삶** ✅ — 기록·감정곡선·캘린더·꽃.
**객체 감지** ⚠️ — 사회스튜디오 1단계로 편입(단독 화면 아님), COCO-SSD 신뢰도 ✅.

## G. 우선순위 — 빠진 기능 채우기 순서

1. **ⓘ 설명 시스템** (교차 · 원본 핵심) — 🔧 진행 중
2. **알고리즘 분석실 7종 + 차트 + 해석 메모 + 리포트** (lab)
3. **펠드먼 4단계 비평** + **도슨트 자동 해설** (critique·work)
4. **데이터 점 제작문서**(메타비평) + Dear Data ⓘ
5. **실제 명화 라이브러리** (퍼블릭 도메인)
6. **교사 큐레이션·취합·AI코치 설정** (admin)
7. 💡 페이지 가이드 · 🌗 테마 토글 · 저장/세션 실동작
