# Claude Code 핸드오프 — 데이터의 눈으로 그림 읽기

> 이 폴더의 디자인(`.dc.html` 프로토타입)을 실제 코드베이스로 옮기기 위한 지시서입니다.
> 함께 보는 문서: **`HANDOFF.md`**(디자인 시스템·토큰·페이지 사양의 단일 기준).

---

## 0. 이 파일들이 무엇인가
- `*.dc.html` = **디자인 프로토타입**(브라우저에서 바로 열리는 인터랙티브 목업, `support.js` 런타임으로 렌더). 프로덕션 코드가 아니라 **시각·동작 사양**입니다.
- 그대로 import해서 쓰는 게 아니라, **여기에 보이는 레이아웃·토큰·상태·인터랙션을 보고** 프로젝트 프레임워크(React/Next/Vue 등)로 구현합니다.
- 색·타이포·간격·컴포넌트 토큰은 `HANDOFF.md` §4에 정리돼 있으니 **그대로 디자인 토큰/테마로 옮기세요.**

## 1. 레포에 넣는 법
1. 이 폴더(다운로드 zip)를 레포에 `design-reference/` 로 추가합니다. 포함: 모든 `*.dc.html`, `support.js`, `image-slot.js`, `HANDOFF.md`, `CLAUDE_CODE_HANDOFF.md`.
2. 각 `.dc.html`은 `design-reference/<파일>.dc.html`을 브라우저로 열어 **실제 동작·디테일을 확인**할 때 사용합니다(개발 중 시각 레퍼런스).
3. 구현은 `src/`(또는 앱 구조)에 새로 작성합니다. `.dc.html`을 프로덕션 번들에 포함하지 마세요.

## 2. Claude Code에 줄 프롬프트 (복사용)
```
design-reference/ 의 .dc.html 프로토타입과 HANDOFF.md 를 디자인 사양으로 삼아
이 제품을 [React + Tailwind / Next.js / 원하는 스택] 으로 구현해줘.

규칙:
- HANDOFF.md §4의 색·타이포·간격 토큰을 theme/tokens로 1:1 이식.
- 테마: 라이트 = 메인·허브·교사대시보드·로그인·내작업 / 다크 = 그림/소리/내삶/사회 분석·작업실·전시.
- 한 화면에 라이트+다크를 섞지 말 것.
- 각 화면의 컴포넌트·상태·인터랙션은 아래 §3 인벤토리를 따를 것.
- 데이터 시각 색 막대/썸네일은 항상 가변 비율(weight)로.
구현 순서: §4 참고. 먼저 공통(토큰·나브·풋터·카드·버튼) → 허브 → 그림분석 → 작업실 → 전시 → 나머지.
```

### 작업실 — 쉬움/전문가(전공) 모드 (중요)
- **쉬움(기본)**: 슬라이더(밀도·흐름·크기·회전·대비·여백) + 옵션(색 소스·배경) + 입력 연결.
- **전문가(`pro:true`)**: 추가 패널 — **레이어**(배경/형태/디테일 on·off), **채널 분리**(색 군집별 on·off → 사용 색 풀 결정), **블렌드 모드**(노멀/스크린/곱하기 → 캔버스 `mix-blend-mode`), **정밀 입력**(시드·밀도 숫자), **내보내기**(1x/2x/4K · PNG/SVG).
- 같은 패턴(쉬움↔전문가 토글 + 고급 패널)을 다른 분석/제작 도구에도 확장 가능.

## 4. 데이터 모델 · 구현 순서
- 스키마는 `HANDOFF.md` §6 참고(Artwork/Recreation/Exhibit/Student). 현재 값은 목업 — 실제 API로 교체.
- 순서: **공통 토큰·컴포넌트 → 허브 → 그림분석 → 작업실(+전문가) → 전시(3D) → 소리/내삶/사회 → AI코치 → 교사대시보드 → 로그인/내작업.**

## 5. 주의
- 3D 전시실은 CSS `perspective` + `transform-style:preserve-3d` + 카메라 `translateZ`. 캡처 도구는 동적 3D를 평면화하므로 실제 브라우저에서 확인.
- 한글 파일명을 영문으로 바꾸려면(`main`,`hub`,`analysis-image`,`analysis-sound`,`analysis-life`,`analysis-society`,`studio`,`gallery`,`coach`,`teacher-dashboard`,`login`,`my-works`) 페이지 간 상대 링크도 함께 수정.
